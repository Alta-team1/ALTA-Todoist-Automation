## How To Run Project
* Open project with **intel IJ Idea**
* Open **src/test/java/starter**
* Right click **CucumberTestSuite**
* Chose **"Run Cucumber TestSuite"**
* Or go to gitbash on terminal
* Type **"mvn clean verify"**
* Click enter
